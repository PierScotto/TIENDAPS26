Importar texto

Una línea por producto. El sistema toma el nombre, el precio y crea el artículo automáticamente.

Cómo lo procesa

El sistema reduce el nombre para que la tarjeta ocupe menos espacio y aplica el cálculo comercial.

  • 1. Toma el precio del mayorista sin IVA.
  • 2. Suma 10% para costo ajustado.
  • 3. Suma 17% sobre ese valor para tu precio final.
  • 4. Convierte NB en categoría Notebook y TAB en Tablet/2 en 1 si aparece.
  • 5. Genera un título corto para la vitrina, manteniendo el detalle completo en el dato original.
